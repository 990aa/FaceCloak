# Universal Adversarial Pixel Poisoning

**Defeating State-of-the-Art Recognition Systems with Verified Imperceptibility.**

[**Live Demo on Hugging Face Spaces**](https://huggingface.co/spaces/a-01a/uacloak)

This toolkit provides a robust, local adversarial pixel poisoning utility designed to disrupt modern visual AI systems while preserving the visual integrity of the original images. Unlike naive noise additions, this project applies **Projected Gradient Descent (PGD)** to attack enterprise-grade visual similarity systems (like FaceNet and CLIP), pushing image embeddings completely out of their semantic clusters without visibly degrading the image (SSIM > 0.98).

## Features

- **Universal Cloaking Capability:** Capable of disguising not just human faces (via FaceNet and CLIP dual-backbone attacks) but also general objects, scenes, and documents (via massive CLIP ViT-L targeting).
- **Verified Imperceptibility:** Strict projection mechanisms mathematically enforce an $L_{\infty}$ or $L_{2}$ budget limit. Image degradation is meticulously monitored using the Structural Similarity Index Measure (SSIM) and Peak Signal-to-Noise Ratio (PSNR) to guarantee visual transparency.
- **Black-Box Transferability:** Perturbations generated using lightweight surrogate models (FaceNet, CLIP ViT-B/32) successfully transfer to computationally massive evaluation "Oracles" (ArcFace, CLIP ViT-L/14) acting as real-world blind benchmarks.
- **Dual-Backbone System:** Uses MTCNN for identity-aware similarity disruption when faces are detected, and universally falls back to CLIP for context-aware geometry destruction on general scenes.
- **Full Benchmarking Suite:** Complete ablation tools and automatic batch testing utilities correctly scope metric drift and confirm the effectiveness of the attacks.

## Evaluator Validation: Black-Box Transferability against State-of-the-Art Oracles

To prove effectiveness without rigging, cloaks generated by our local surrogate models are benchmarked against independent, state-of-the-art Oracle models (`ArcFace` for faces, `CLIP ViT-L/14` for general images). 

A successful evasion here represents a genuine failure of commercial-grade recognition systems, not an artifact of a weak local evaluator. Our attack, generated on lightweight surrogates, successfully transfers to completely blind both ArcFace and CLIP ViT-L/14, demonstrating that the cloaking effectively defeats commercial-grade recognition systems while maintaining an SSIM greater than 0.98.

## Usage 

Dependencies are managed using `uv`. 

```bash
uv sync
```

To run the interactive Gradio dashboard locally:

```bash
uv run app.py
```

To run the robust offline Phase 14 benchmarking suite:

```bash
uv run eval.py --manifest benchmarks/sample_manifest.csv 
```

To conduct sweeping Phase 13 hyperparameter ablations:

```bash
uv run ablation.py 
```

## Privacy & Security 

This tool is entirely open-source and local-first. When running through the local interface or Hugging Face Space, zero telemetry, logging, or third-party API keys are utilized.
